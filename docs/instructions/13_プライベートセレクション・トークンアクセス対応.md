# 指示文 13 — プライベートセレクション トークンアクセス対応

## 概要

マジックリンク（`/private-selection?token=xxx`）経由で
ログインなしにプライベートセレクションを閲覧できるようにする。

**アクセス制御の全体像:**
```
ルート1: ログイン済み会員 → そのまま閲覧
ルート2: ?token=xxx → Admin API で検証 → 有効なら閲覧
ルート3: 未ログイン・トークンなし → /members/login へリダイレクト
```

## 確認済みのAPI仕様

```
GET /api/private-selection/verify?token=<token>

成功: { valid: true, customerId: string, email: string, expiresAt: string }
失敗: { valid: false, reason: "not_found" | "expired" | "no_token" | "server_error" }
```

## Step 1: lib/api.ts にトークン検証APIを追加

```typescript
// lib/api.ts への追記

export interface TokenVerifyResult {
  valid: true;
  customerId: string;
  email: string;
  expiresAt: string;
} | {
  valid: false;
  reason: "not_found" | "expired" | "no_token" | "server_error";
}

export async function verifyPrivateSelectionToken(
  token: string
): Promise<TokenVerifyResult> {
  try {
    return await fetchFromAdmin<TokenVerifyResult>(
      `/api/private-selection/verify?token=${encodeURIComponent(token)}`,
      { next: { revalidate: 0 } }
    );
  } catch {
    return { valid: false, reason: "server_error" };
  }
}
```

## Step 2: middleware.ts を更新

トークンがある場合は middleware でリダイレクトしないようにする。

```typescript
// middleware.ts を以下に全面更新
import { withAuth } from "next-auth/middleware";
import { NextResponse } from "next/server";
import type { NextRequest } from "next/server";

// トークンがある場合は認証チェックをスキップ
export default withAuth(
  function middleware(req: NextRequest) {
    return NextResponse.next();
  },
  {
    callbacks: {
      authorized: ({ req, token }) => {
        const pathname = req.nextUrl.pathname;

        // /private-selection はトークンがあれば認証不要
        if (pathname.startsWith("/private-selection")) {
          const searchToken = req.nextUrl.searchParams.get("token");
          // cookie にトークンを保存している場合も考慮
          const cookieToken = req.cookies.get("ps_token")?.value;
          if (searchToken || cookieToken) return true;
          return !!token; // ログイン済みなら通す
        }

        // それ以外の保護ルートはログイン必須
        return !!token;
      },
    },
  }
);

export const config = {
  matcher: [
    "/members/mypage/:path*",
    "/members/favorites/:path*",
    "/members/inquiries/:path*",
    "/private-selection/:path*",
  ],
};
```
